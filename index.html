Research

My research interests lie in social data processing, natural language processing, human-computer interactions.

ME-Prune: Highly Compressed and Robust Neural Networks via Matrix Entropy Based Channel Pruning
A. Zhao, Y. Yang, E. Ye, Z. Liu, X. Yue, V. Shirsat, Y. Zhou, K. Keutzer, J. Gonzalez, K. Ramchandran, M. Mahoney.
IEEE/CVF Conference on Computer Vision and Pattern Recognition, 2022 (in submission)

ME-Prune prunes a neural network from an out-of-distribution (OOD) robustness perspective, focusing on mitigating classification errors that result from common everyday corruptions modeled by data augmentation schemes. We perform network structure search using matrix entropy (ME), a random matrix theory metric that is normally applied in statistical physics, rendering it a novel application in pruning.

NewsPod: Automatic and Interactive News Podcasts
P. Laban, E. Ye, S. Korlakunta, J. Canny, M. Heart.
ACM Conference on Intelligent User Interfaces, 2022 (in submission)

NewsPod, an interactive and automated news podcast, organizes news content into a simulated conversation in which several automated voices ask and answer questions about a topic. The listener can at any point join the conversation by asking a question, which the system attempts to answer automatically. We evaluated the Q&A narrative style and participant interactions; more than 80% of the participants said they would use NewsPod to stay informed, and 85% of listeners ask questions and interact with the podcast. Participant-prompted questions reveal current limitations in natural language processing on neural network-based question-answering models.
